# Simple Network Scanner (ARP)
Built as part of my Security Technology portfolio at MMU.

## Description
This tool uses the Scapy library to send ARP requests across a local network to discover active devices and their MAC addresses.

## Prerequisites
* Python 3.x
* Scapy (`pip install scapy`)
* Npcap (for Windows users)

## How to Run
1. Run VS Code or Terminal as Administrator.
2. Update the IP range in the script.
3. Run: `python scanner.py`
